# Xarpite ドキュメント

このディレクトリには、XarpiteのGitHub Pagesドキュメントが含まれています。

## デザインコンセプト

### カラースキーム
- **プライマリカラー**: `#E50000` (明るい赤)
- **ダークカラー**: `#860000` (暗い赤)
- プロジェクトアイコンのグラデーションカラーを使用

### デザイン特徴
- **モダン**: クリーンで現代的なレイアウト
- **エレガント**: 適度な余白とシャドウによる奥行き
- **プロフェッショナル**: 読みやすいタイポグラフィ
- **レスポンシブ**: モバイルからデスクトップまで対応
- **アクセシブル**: 高コントラストで読みやすい

## 構造

```
docs/
├── _config.yml           # Jekyll設定
├── _layouts/             # レイアウトテンプレート
│   └── default.html      # デフォルトレイアウト
├── assets/               # 静的アセット
│   ├── css/
│   │   └── style.css     # メインスタイルシート
│   └── images/           # 画像ファイル
├── index.md              # トップページ
└── ja/                   # 日本語ドキュメント
    ├── index.md
    ├── syntax.md
    └── ...
```

## ローカルでの確認

```bash
cd docs
bundle install
bundle exec jekyll serve
```

ブラウザで `http://localhost:4000/xarpite/` を開いてください。

## デプロイ

GitHub Actionsが自動的にGitHub Pagesにデプロイします。
`main`ブランチへのプッシュ時に自動実行されます。
