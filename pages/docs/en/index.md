# Xarpite Documentation

Xarpite is an interpreted language designed for one-liners.

This documentation explains Xarpite's syntax, features, and usage.

## Documentation Structure

Please refer to each topic from the menu on the left.

- **Basic Syntax**: Syntax, statements and expressions, whitespace and comments, variables, etc.
- **Data Types**: Numbers, strings, regular expressions, BLOBs, arrays, objects, etc.
- **Control Structures**: Booleans and conditionals, return and error handling, control structures, etc.
- **Operators**: Comparison, property access, stream/assignment/pipe operators, etc.
- **Built-in Features**: Built-in constants and functions, data conversion functions, etc.
- **Advanced Topics**: Samples, command-line tools, JavaScript version, coroutines, JSON manipulation, etc.

## Getting Started

To get started with Xarpite, please refer to the [Syntax](syntax.html) page first.
