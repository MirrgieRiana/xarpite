# Xarpiteドキュメント

Xarpiteはワンライナー向けに設計されたインタープリター型言語です。
ストリームとパイプ演算子を中心に、入力から変換、出力までを短い記述でつなげられるのが特徴です。
CLI/ブラウザ/ネイティブと複数の実装で同一の言語仕様を提供し、どの環境でも同じコードで動作します。
このドキュメントでは、基本文法から組み込み関数、実行環境ごとの機能までを章立てで解説します。
