# マイグレーションガイド

このガイドは、既存のスクリプトを新しいAPIバージョンに追従させるための、書き換えの手引きです。

---

このガイドは、APIバージョンごとに節を設け、その世代で導入された破壊的変更ごとに、既存の記法やイディオムをどのように書き換えればよいかを説明します。

## APIバージョン5

### `FILES` の戻り値にディレクトリのパスが含まれるようになりました

APIバージョン4までの `FILES` は、ディレクトリ直下のファイル名のみを返していました。APIバージョン5では、`dir` を先頭に含むパスを返すようになりました。

従来の挙動を維持するには、`FILES` の呼び出しを `FILE_NAMES` に置き換えてください。`FILE_NAMES` は、APIバージョンに依らず、従来の `FILES` と同じくファイル名のみを返します。

変更前:

```
FILES("dir")
```

変更後:

```
FILE_NAMES("dir")
```
