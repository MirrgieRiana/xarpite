# マイグレーションガイド

このガイドは、既存のスクリプトを新しいAPIバージョンに追従させるための、書き換えの手引きです。

---

このガイドは、APIバージョンごとに節を設け、その世代で導入された破壊的変更ごとに、既存の記法やイディオムをどのように書き換えればよいかを説明します。

## APIバージョン5

### `#` による行コメントの開始位置

APIバージョン5以降では、 `#` による行コメントは、行頭、もしくはスペースやタブの直後でのみ開始できます。

APIバージョン4以前で、直前に空白を挟まずに `#` から行コメントを始めていた箇所では、`#` の前にスペースを補ってください。
