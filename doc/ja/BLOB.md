# BLOB

BLOBはバイナリデータを高速に扱うための型で、一部のAPIが要求します。

BLOBが格納するバイト列は、符号なし8bit整数として管理されます。

## 配列からのBLOBの生成

BLOBは `BLOB.new(array)` で生成します。

`array` は0個以上の数値からなる配列でなければならず、変換時に小数点以下の四捨五入と下位8ビットの切り捨てが行われます。

```shell
$ xa 'BLOB.of([1, 0, -1, 256, 257, 1.4, 1.6])'
# [1;0;255;0;1;1;2]
```

## 配列への変換

BLOBは `blob::to_array()` で直接配列に変換できます。

```shell
$ xa 'BLOB.of([1, 2, 3])::to_array()'
# [1;2;3]
```
